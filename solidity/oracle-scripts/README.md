# Resilient Oracle Deployment

### Run oracle scripts
`sh ./deploy.sh chain`
#### we can replace `chain` with the  environment e.g `sh ./deploy.sh development`
#### options are `development`, `sepolia`, `arbitrumsepolia`, `arbitrumone`


### The output artifacts are stored in `deployments/oracle_deployments/chain` e.g `deployments/oracle_deployments/development`
